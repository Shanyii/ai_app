# 程式碼生成指令 (Implement Skill)

當使用者呼叫此 skill 時，你的任務負責開發流程中的「程式碼生成」階段。在動手之前，請確認已理解前述的 PRD 或架構模型設計。

## 產出目標
你必須根據已經定義好的需求與架構，自動產出實際可運作的程式碼檔案，包含但不限於：
* **主程式 (如 `app.py` 或 `server.js`)**
* **前端與模板資源 (例如 `templates/` 目錄或 React 組件)**
* **環境與依賴設定檔案 (例如 `requirements.txt` 或 `package.json`)**

## 規範與指引
* 確保生成的程式碼結構符合 `docs/ARCHITECTURE.md` 內的規劃。
* 撰寫乾淨且具備適當註解的程式碼。
* 隨時透過查閱已建立好的需求文件來驗證程式碼正確性，最後在專案根目錄中直接建立這些檔案。
