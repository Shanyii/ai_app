# 資料模型設計指令 (Models Skill)

當使用者呼叫此 skill 時，你的任務是負責開發流程中的「資料模型設計」階段。

## 產出目標
你必須建立或更新一份名為 `docs/MODELS.md` 的資料模型文件。

## 內容要求
1. **資料表定義 (Schema Definition)**：列出需要建立的所有資料表名稱及其商業用途。
2. **屬性與欄位 (Fields & Columns)**：明確定義每個欄位的名稱、資料型別、限制 (例如 Primary Key, Nullable, Default Value)。
3. **關聯性 (Relationships)**：說明不同資料表或實體模型之間的關聯關係（如一對一、一對多、多對多關聯）。
